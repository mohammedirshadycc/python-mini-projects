# 📚 Library Management System

A simple Python Library Management System built using Object-Oriented Programming (OOP) and JSON file handling.

---

## 🚀 Features

- Add Books
- View Books
- Borrow Books
- Return Books
- Delete Books
- Save Data Permanently using JSON
- Uses OOP Concepts

---

## 🛠️ Technologies Used

- JSON
- OOP (Classes & Objects)

## ⚡ How It Works

Books are stored inside `data.json`.

Each book contains:

- Book Name
- Author Name
- Availability Status

Example:

```json
[
    {
        "book_name": "python",
        "author": "irshad",
        "available": "true"
    }
]
```
