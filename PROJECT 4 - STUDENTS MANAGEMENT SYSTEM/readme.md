# 🎓 Student Management System

A simple Python Student Management System that allows users to manage student records using JSON file storage.

---

# 🚀 Features

✅ Add new students
✅ View all students
✅ Search students using roll number
✅ Update student marks
✅ Delete student records
✅ Permanent data storage using JSON
✅ Input validation using try-except
✅ Object-Oriented Programming using classes

---

# 🛠 Technologies Used

* Python
* JSON
* OOP (Classes & Objects)
* File Handling
