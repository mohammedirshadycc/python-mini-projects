# Bank Management System 🏦

A simple Python Bank Management System that allows users to:

* Create accounts
* Login to existing accounts
* Deposit money
* Withdraw money
* Check balance
* Delete accounts
* Save account data permanently using JSON

---

# Features 🚀

✅ Create multiple bank accounts
✅ Login system for users
✅ Deposit money safely
✅ Withdraw money with balance checking
✅ Check account balance anytime
✅ Delete accounts
✅ Data saved permanently in `data.json`

# How the Program Works ⚙️

1. User creates or logs into an account
2. Account data is loaded from `data.json`
3. User can:

   * Deposit money
   * Withdraw money
   * Check balance
   * Delete account
4. Data is automatically saved
